RG To NFA

Instructions
1. Input regular grammar (example: S - aS | ε )
2. Each sentence is separated by line break (Enter or "\n")
3. State must be one uppercase alphabet [A-Z]
4. Transition variable can be number [0-9] or lowercase alphabet [a-z]
5. First sentence will be an initial state (entry point)
6. Sentence arrow is denoted by a hyphen ("-") ex. S -> aS (❌), S - aS (✔️) [Will change later]
7. Multiple transition variable is denoted by a pipe ("|") ex. aS | bA | abB
8. No space for each transition ex. S - a S (❌), S - aS (✔️)

How To Use
1. Input regular grammar sentences on top textbox
2. Click Parse
3. Input strings to be checked (Currently supports only one sentence)
4. Click Check and the result will be shown below (True for accepted and false for rejected).
